# How to Study?

## Problem Solving

- [ ] Attempt solo (10–20 min).
- [ ] If stuck, look up the solution (AI or official).
- [ ] Annotate the solution with comments until you understand it.
- [ ] Re-implement from scratch without looking (10–20 min).

## Algorithms & Data Structures

- [ ] Study the concept: explanation, example problem, and solution.
- [ ] Annotate the solution with comments until you understand it.
- [ ] Re-implement from scratch without looking (10–20 min).
- [ ] Solve similar problems to reinforce.
